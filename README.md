# bb-stats

Player count statistics dashboard for the Blood Bound Squad server, powered by the BattleMetrics API.

## Features

- 28-day hourly player count history
- Lines grouped by weekday for pattern comparison
- Current hour marker
- Hourly data refresh with 24-hour localStorage persistence

## Development

```bash
npm install
npm run dev
```

## Deploy

Automatically deploys to GitHub Pages on every push to `main`.

Live: https://PD-22.github.io/bb-stats/
